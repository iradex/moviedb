Documentation - MovieDB API

Description Endpoint Method Type Query Parameters Request Data Format Response Data Format
Returns list of all movies. /movies GET - - A JSON Object with data about all the movies
Returns information about a single movie /movies/[Title] GET [Title] is the title of the movie for which the details will be responded. - A JSON Object containing details about the movie that was requested via the endpoint. Example:
{ title: "Forrest Gump", actor: "Tom Hanks", length: 186 }
Returns a list of movies of a certain genre. /movies/genres/genre GET [Genre] is the genre of the movies for which the list will be generated. - A JSON Object containing a list of movie titles of a certain genre.
Returns information about a certain director /directors/[director’s name] GET [Director’s name] is the name of the director for which the information will be displayed. - A JSON Object containing the information about the particular director.
Creates a new user /users POST - A JSON Object holding the name of the user as well as the password. Example:
{ username: „MovieFriend“, Password: „12345“, email: „moviefriend@gmail.com", dob: 14011991 }		 A message that the user was successfully created
Updates user data /users/[username] PUT [Username] is the username of the current user… A JSON Object holding the name of the user as well as his key data. Example:
{ username: „MovieFriend“, Password: „p6iasd86ew“, email: „moviefriend@gmail.com", dob: 14011991 } 		A message that the user data was updated successfully.
Lets users add a list to their personal movie list /users/[username]/favorites POST [Username] is the username of the current user… A JSON Object holding the name of the movie that should be added. Example:
{ title: „Avatar“ }		 A message showing that the title was added succesfully.
Lets users delete a movie from their favorites list /users/[username]/favorites/[title] DELETE [Username] is the username of the current user. [title] is the name of the movie that should be deleted. - A message showing that the title was successfully removed from the favorites list.
Lets users delete their account. /users/[username] DELETE [Username] is the username of the current user… - A message showing that the user was successfully deleted.
